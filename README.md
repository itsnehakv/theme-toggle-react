A simple React project that allows users to switch between **light and dark modes**.

### Features
- **Light & Dark Modes**: Switch between themes easily.
- **Fluid Animations**: Smooth transitions when toggling themes.
- **Persistent Settings**: Saves your preferred theme in `localStorage`.
- **Dynamic Styles**: Changes font and link colors based on the selected theme.
- **Responsive Design**: Works on all device sizes.
- **First-time Netlify Deployment**: Fully deployed online, no server setup needed.
